# TVRemote Plugin for Jeedom

## 🎯 Description

Plugin Jeedom pour contrôler les appareils Android TV via deux protocoles :
- **AndroidTVRemote2** : Protocole Google TV Remote (recommandé)
- **ADB** : Android Debug Bridge (pour compatibilité maximale) ✨ **NOUVEAU**

## ✨ Nouveauté : Support ADB

Le plugin supporte désormais le protocole ADB (Android Debug Bridge) en complément d'AndroidTVRemote2 !

### Avantages d'ADB :
- ✅ Compatible avec TOUS les appareils Android (pas seulement Android TV)
- ✅ Accès aux commandes shell avancées
- ✅ Contrôle système complet
- ✅ Automatisation avancée
