# TVRemote Plugin for Jeedom

## 🎯 Description

Plugin Jeedom pour contrôler les appareils Android TV via deux protocoles complémentaires :

### 📱 AndroidTVRemote2 (Recommandé)
Protocole officiel Google TV Remote pour un contrôle simple et efficace :
- ✅ Connexion sécurisée avec appairage
- ✅ Navigation intuitive (HAUT, BAS, GAUCHE, DROITE, OK)
- ✅ Contrôle multimédia (PLAY, PAUSE, STOP)
- ✅ Gestion du volume et du power
- ✅ Saisie de texte à distance
- ✅ Faible latence et stable

**📚 Documentation :**
- [Bibliothèque Python androidtvremote2](https://github.com/tronikos/androidtvremote2)
- [Documentation API Google TV](https://developers.google.com/android/reference/com/google/android/gms/cast/tv/TvRemoteManager)

### 🔧 ADB (Android Debug Bridge)
Protocole avancé pour un contrôle système complet :
- ✅ Accès aux commandes shell avancées
- ✅ Contrôle système approfondi
- ✅ Automatisation complexe
- ✅ Installation d'applications
- ✅ Gestion des paramètres système

**📚 Documentation :**
- [Documentation officielle ADB](https://developer.android.com/tools/adb)
- [Commandes ADB courantes](https://developer.android.com/tools/adb#am)
- [Guide ADB Shell](https://adbshell.com/)

## 🚀 Fonctionnalités principales

- 🎮 **Contrôle complet de la navigation** : Touches directionnelles, retour, home, menu
- 📺 **Gestion multimédia** : Play, pause, stop, avance/retour rapide
- 🔊 **Contrôle du volume** : Augmentation, diminution, mute
- ⚡ **Power management** : Allumage, extinction, veille
- ⌨️ **Saisie de texte** : Envoi de texte pour recherche ou formulaires
- 🚀 **Lancement d'applications** : Ouverture directe d'applications Android
- 🔍 **Commandes shell ADB** : Exécution de commandes personnalisées (mode ADB)
- 📱 **Multi-appareils** : Gestion de plusieurs Android TV simultanément
- 🔐 **Connexion sécurisée** : Appairage avec code PIN (AndroidTVRemote2)
- 🎯 **Interface Jeedom** : Intégration native dans les scénarios et dashboard
